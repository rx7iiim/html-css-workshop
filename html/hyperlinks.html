HTML hyperlinks (anchor links)

What is a hyperlink?

A hyperlink connects one page (or resource) to another. Use the <a> tag with the href attribute.

Absolute URL

Rendered example:

Go to example.com

Source (shown as text):

<a href="https://example.com">Go to example.com</a>

Relative URL

Rendered example (assumes the file exists in your project):

Open structure.html (relative)

Source:

<a href="../structure.html">Open structure.html (relative)</a>

Fragment (page anchor)

Click to jump to a section below:

Jump to section

Section target

This is the destination for the fragment link above.

Source for the fragment link:

<a href="#section-target">Jump to section</a> ... <h3 id="section-target">Section target</h3>

Email and telephone links

Send an email:

Email someone@example.com

Call a phone number (works on devices that support calling):

Call +1 234 567 890

Source:

<a href="mailto:someone@example.com">Email someone</a>

<a href="tel:+1234567890">Call number</a>

Open in a new tab

Rendered example (opens in a new tab/window):

Open example.com in a new tab

Source:

<a href="https://example.com" target="_blank">Open example.com in a new tab</a>

Link text and title

Example (has title)

Source:

<a href="https://example.com" title="Visit example">Example (has title)</a>

Accessibility tips

  • Use clear, descriptive link text (avoid "click here").
  • If a link opens a new tab, let users know (for example "(opens in a new tab)").

These examples demonstrate common hyperlink uses in plain HTML.
