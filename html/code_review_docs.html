HTML for Beginners

Your Complete Guide to Building Websites

1What is HTML?

HTML stands for HyperText Markup Language. It's the foundation of every website you visit. Think of HTML as the skeleton or blueprint of a webpage - it defines the structure and content.

üè† Real-World Analogy

Imagine building a house:

  • HTML = The structure (walls, rooms, doors)
  • CSS = The decoration (paint, furniture, style)
  • JavaScript = The functionality (lights, plumbing, electricity)

2Why Learn HTML?

  • Easy to learn - You can create your first webpage in minutes
  • Foundation of web development - Required for any web career
  • Universal - Works on every device and browser
  • No special tools needed - Just a text editor and a browser
  • Free and open - No licenses or costs

3How HTML Works

HTML uses tags to tell the browser what each piece of content is. Tags are like labels that wrap around content.

Basic Structure:

<tagname>Your content goes here</tagname>

Key Concepts:

  1. Opening tag: <tagname> - Starts the element
  2. Content: The text or other elements inside
  3. Closing tag: </tagname> - Ends the element (note the forward slash)

Example:

<h1>Welcome to My Website</h1>

This tells the browser "this is a main heading"

4Essential HTML Tags

Structure Tags

<html> - Wraps the entire webpage (the root of every HTML document)

<head> - Contains metadata (information about the page, not visible to users)

<body> - Contains everything visible on the page

Basic Page Structure:

<html>
  <head>
    <title>My Page Title</title>
  </head>
  <body>
    Content goes here!
  </body>
</html>

Text Tags

Headings - <h1> through <h6>

  • <h1> is the biggest/most important
  • <h6> is the smallest
  • Use for titles and section headers
<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Smaller heading</h3>

Paragraphs - <p>

For regular text blocks

<p>This is a paragraph of text.</p>

Text Formatting

  • Bold - <strong> or <b>
  • Italic - <em> or <i>
  • Line Break - <br> (self-closing tag)

Links - <a>

Creates clickable links (anchor tag). Uses the href attribute to specify where to go.

<a href="https://www.google.com">Click here to visit Google</a>
üí° Pro Tips:
  • Email link: <a href="mailto:someone@example.com">Email me</a>
  • Open in new tab: <a href="..." target="_blank">Link</a>

Images - <img>

Displays images (self-closing tag)

<img src="photo.jpg" alt="A beautiful sunset">
  • src attribute = image location
  • alt attribute = description (important for accessibility)

Lists

Unordered List (bullets)

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

Ordered List (numbers)

<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>

Containers

  • <div> - Generic container for grouping content (like an invisible box)
  • <span> - Inline container for small portions of text

5Attributes Explained

Attributes provide additional information about HTML elements. They go inside the opening tag.

Syntax:

<tagname attribute="value">Content</tagname>

Common Attributes:

  • id - Unique identifier for an element
  • class - Groups elements for styling
  • style - Inline CSS styling
  • href - URL for links
  • src - Source for images/media
  • alt - Alternative text for images

6Creating Your First HTML Page

Step 1: Open a Text Editor

  • Windows: Notepad
  • Mac: TextEdit
  • Better options: VS Code, Sublime Text, Atom

Step 2: Write Your HTML

<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
    <p>I'm learning HTML and it's fun!</p>
</body>
</html>

Step 3: Save the File

  • Save as index.html (or any name ending in .html)
  • Make sure to select "All Files" if using Notepad

Step 4: Open in Browser

  • Double-click the file
  • Or right-click ‚Üí Open With ‚Üí Your browser

7Important Concepts

1. Nesting

Elements can go inside other elements (like Russian dolls)

Correct

<div>
  <p>Inside div</p>
</div>

Incorrect

<div>
  <p>Wrong</div>
</p>

2. Self-Closing Tags

Some tags don't have content, so they don't need a closing tag:

  • <br> - Line break
  • <img> - Image
  • <hr> - Horizontal rule
  • <input> - Form input

3. Comments

Comments are notes in your code that browsers ignore:

<!-- This is a comment -->
<!-- 
  This is a 
  multi-line comment
-->

8Common Beginner Mistakes

Forgetting closing tags

<p>Never ends
<p>Confusing

Always close tags

<p>Proper ending</p>
<p>Much better</p>

Wrong nesting order

<strong><em>Wrong</strong></em>

Correct nesting

<strong><em>Right</em></strong>
