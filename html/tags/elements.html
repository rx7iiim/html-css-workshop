HTML elements — explained with examples

Small, focused examples that demonstrate what an HTML element is, how tags and attributes work, and common element types.

What is an element?

An HTML element is the building block of a web page. An element usually consists of a start tag, content, and an end tag:

Rendered

Welcome!

This is a simple paragraph inside an <h2> and <p> element.

<

Start tag: <h2>, end tag: </h2>. The tag name (h2) is the element's type.

Tags vs elements vs attributes

A tag is the markup you write (for example <a>). An element is the combination of the start tag, content, and end tag. Attributes give additional information about an element (for example href on an anchor).

Example: 
<a href="https://example.com" target="_blank">Visit example.com</a>

Rendered:

Visit example.com
