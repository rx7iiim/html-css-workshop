HTML images — basic examples

The <img> tag

The <img> tag displays an image. Use src for the image URL and alt for alternative text.

Simple example

<img src="path/to/image.jpg" alt="Description of image">

Inline SVG as image source

SVG data can be embedded directly in the src using a data URL (advanced but useful for tiny images):

<img src="data:image/svg+xml;utf8,<svg ...>...</svg>" alt="inline svg">

Width and height

You can set width and height attributes (in pixels). Modern practice often prefers CSS, but attributes are simple and valid.

<img src="path/to/image.jpg" alt="desc" width="200" height="120">

Alt text guidance

  • If the image conveys meaning, provide descriptive alt text.
  • If the image is purely decorative, set alt equal to empty (alt="") so screen readers skip it.

Image captions

To add a caption for an image, place a short descriptive paragraph after the image. This helps users understand the image's context.
